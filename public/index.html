How It Works

Four simple steps to optimize your Excel formulas with AI

1

Select Formula

Select a cell with a formula in Excel or highlight the formula text. The add-in automatically detects its content.

2

Press Ctrl+Shift+O

Use the hotkey or click "Optimize" in the task pane. Your formula is sent for analysis via encrypted connection.

3

AI Analyzes

GPT-4, Claude, or Gemini analyzes your formula: finds redundant calculations, optimizes structure, suggests more efficient functions.

4

Get Results

The optimized formula appears in the panel with detailed explanation of changes. You decide whether to apply it or not.

üîç What exactly is transmitted?

When you optimize a formula like =SUM(A1:A10)*2, we only send the formula text to AI. Values from cells A1:A10 are NOT transmitted. AI only sees the formula structure and suggests optimizations, for example: =SUMPRODUCT(A1:A10,2). Your data stays in Excel.

Installation

Get started in minutes

For End Users

  1. Open Microsoft Excel (version 2016 or newer)
  2. Go to Insert ‚Üí Add-ins ‚Üí My Add-ins
  3. Select Add from File
  4. Enter URL: https://funkydonkey.github.io/blade/manifest.xml
  5. Click Upload and confirm installation
  6. Open Home ‚Üí Show Optimizer
  7. In settings, enter your API key from OpenAI, Anthropic, or Google
  8. Done! Use Ctrl+Shift+O to optimize formulas

For Developers

  1. Clone repository: git clone https://github.com/funkydonkey/blade.git
  2. Install dependencies: npm install
  3. Run dev server: npm start
  4. Sideload manifest.xml in Excel
  5. For backend: cd backend && npm install && npm run dev

Frequently Asked Questions

Common questions about the Excel AI Optimizer

Is it free?

Yes, the add-in is completely free and open source (MIT License). You only need an API key from an AI provider (OpenAI, Anthropic, Google), which you pay for separately according to their pricing. Or use Ollama locally for free.

Is it safe to send formulas to AI?

Only formula text is transmitted, without cell data. If formulas contain confidential business logic, we recommend self-hosting the backend or using local Ollama. All requests are protected with HTTPS encryption.

Which AI models are supported?

OpenAI GPT-4 Turbo, Anthropic Claude 3.5 Sonnet, Google Gemini Pro, and local models via Ollama. You can choose the provider in settings.

Does it work offline?

An internet connection is required, except when using local Ollama. In that case, you can run both backend and AI model locally.

Can I use it in a corporate environment?

Yes. For corporate use, we recommend deploying the backend on your infrastructure and setting up centralized installation via Microsoft 365 Admin Center. Instructions are available in the documentation.

Where are my API keys stored?

API keys are stored only in your browser's localStorage. They are never sent to our backend - only directly to your chosen AI provider.

How can I contribute to the project?

We welcome all contributions! Create Issues with suggestions, submit Pull Requests with improvements. All code is on GitHub.
