How It Works

AI-powered optimization and instant local formatting in just a few clicks.

Option 1: Optimize with AI

1

Select Formula

Select a cell with a formula in Excel. The add-in automatically detects its content.

2

Click "Optimize with AI"

Or use hotkey Ctrl+Shift+O. Your formula is sent for analysis via encrypted HTTPS connection.

3

AI Analyzes

GPT-4, Claude, or Gemini analyzes your formula: finds redundant calculations, optimizes structure, suggests more efficient functions.

4

Review & Apply

The optimized formula appears with detailed explanation. You decide whether to apply it or not.

Option 2: Make Readable (No AI)

1

Select Formula

Select a cell with a formula in Excel that you want to format.

2

Click "Make Readable"

Click the "Make Readable (No AI)" button. No internet required‚Äîworks instantly!

3

Instant Formatting

Your formula is formatted locally with proper indentation and line breaks. No API calls, no waiting.

4

Apply or Copy

Review the formatted formula and apply it to your cell or copy it to clipboard.

Get Your API Key

Choose an AI provider and get your API key to start optimizing formulas

OpenAI GPT-4

Most popular choice with excellent formula optimization capabilities.

  1. Go to platform.openai.com/signup
  2. Create an account or sign in
  3. Navigate to API keys section
  4. Click Create new secret key
  5. Copy and save your key securely

üí° Cost: Pay-as-you-go, ~$0.01-0.03 per optimization

Anthropic Claude

Advanced reasoning and detailed explanations for complex formulas.

  1. Visit console.anthropic.com
  2. Sign up for an account
  3. Go to API Keys
  4. Click Create Key
  5. Name it and copy the key

üí° Cost: $5 free credits, then pay-as-you-go

Google Gemini

Free tier available with generous limits for personal use.

  1. Go to Google AI Studio
  2. Sign in with Google account
  3. Click Get API key
  4. Create key for new or existing project
  5. Copy your API key

‚úì Free: 60 requests per minute at no cost

Ollama (Local)

Run AI models locally on your computer. Completely free and private.

  1. Download from ollama.ai
  2. Install and run Ollama
  3. Pull a model: ollama pull llama2
  4. Set backend endpoint to localhost
  5. No API key needed!

‚úì Free: Unlimited, runs on your hardware

üîê Keep Your API Key Safe

Never share your API key publicly. It's stored only in your browser and sent directly to the AI provider. If you accidentally expose it, regenerate a new key immediately from your provider's dashboard.

Installation

Get started in minutes

For normal human beings

  1. Open Microsoft Excel (version 2016 or newer)
  2. Go to Home ‚Üí Add-ins ‚Üí Other Add-ins ‚Üí My Add-ins
  3. Select Add from File
  4. Download this file or just enter URL: https://funkydonkey.github.io/blade/manifest.xml
  5. Click Upload and confirm installation
  6. Open Home ‚Üí Show Optimizer
  7. In settings, enter your API key from OpenAI, Anthropic, or Google
  8. Done! Use Ctrl+Shift+O to optimize formulas

For people who afraid of 5G

  1. Clone repository: git clone https://github.com/funkydonkey/blade.git
  2. Install dependencies: npm install
  3. Run dev server: npm start
  4. Sideload manifest.xml in Excel
  5. For backend: cd backend && npm install && npm run dev

Frequently Asked Questions

Common questions about the Excel AI Optimizer

Is it free?

Yes, the add-in is completely free and open source (MIT License). You only need an API key from an AI provider (OpenAI, Anthropic, Google), which you pay for separately according to their pricing. Or use Ollama locally for free.

Is it safe to send formulas to AI?

Only formula text is transmitted, without cell data. If formulas contain confidential business logic, we recommend self-hosting the backend or using local Ollama. All requests are protected with HTTPS encryption.

Which AI models are supported?

OpenAI GPT-4 Turbo, Anthropic Claude 3.5 Sonnet, Google Gemini Pro, and local models via Ollama. You can choose the provider in settings.

Does it work offline?

An internet connection is required, except when using local Ollama. In that case, you can run both backend and AI model locally.

Can I use it in a corporate environment?

It depends on your local IT policy. In some companies xml add-ins can be blocked. If that the case - you can run the backend locally or ask for setting up centralized installation via Microsoft 365 Admin Center. Instructions are available in the documentation.

Where are my API keys stored?

API keys are stored only in your browser's localStorage. They are never sent to our backend - only directly to your chosen AI provider.

How can I contribute to the project?

We welcome all contributions! Create Issues with suggestions, submit Pull Requests with improvements. All code is on GitHub.
